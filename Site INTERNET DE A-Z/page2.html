UX vs UI

UX vs UI UX and UI: Two terms that are often used interchangeably, but actually mean very different things. UX design refers to the term “user experience design”, while UI stands for “user interface design”. Both elements are crucial to a product and work closely together UX design is all about identifying and solving user problems; UI design is all about creating intuitive, aesthetically-pleasing, interactive interfaces. UX design usually comes first in the product development process, followed by UI. The UX designer maps out the bare bones of the user journey; the UI designer then fills it in with visual and interactive elements.
